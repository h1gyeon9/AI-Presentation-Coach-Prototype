# AI Presentation Coach Prototype

발표/면접 영상을 AI가 분석하는 경험을 보여주기 위한 정적 MVP입니다.

## 현재 범위

- 실제 AI API 요청 없음
- 브라우저 카메라 프리뷰 사용
- 영상 위 얼굴 위치, 시선 방향, 자세 포인트 분석 오버레이 표시
- 분석 진행 단계 시뮬레이션
- 정적으로 준비된 샘플 발표 피드백 리포트 출력

## 실행 방법

별도 서버나 빌드 과정 없이 `index.html`을 브라우저에서 열면 됩니다.

Netlify에 배포한 경우에는 변경사항을 Git에 push하면 정적 사이트로 그대로 동작합니다.

## 데모 흐름

1. `카메라 시작`을 눌러 영상 프리뷰와 분석 오버레이를 확인합니다.
2. `분석 리포트 생성`을 누르면 얼굴 위치 감지, 시선 추정, 자세 분석 단계가 진행됩니다.
3. 완료 후 샘플 코칭 리포트가 출력됩니다.

카메라 권한이 없어도 오버레이와 리포트 시뮬레이션은 동작합니다.
